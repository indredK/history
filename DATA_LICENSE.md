# 数据许可证 (Data License)

## 概述

本项目涉及两类主要组件，分别采用不同的许可证：

1. **代码**: MIT License（见 [LICENSE](./LICENSE)）
2. **数据**: CC-BY-4.0（本文档）

## 数据许可 - 知识共享署名 4.0 国际版 (CC-BY-4.0)

### 许可原文

本项目的所有数据文件（包括历史事件、人物信息、地理边界等）采用 **Creative Commons Attribution 4.0 International (CC-BY-4.0)** 许可。

完整法律文本可在以下地址查阅：https://creativecommons.org/licenses/by/4.0/legalcode

### 简易说明

CC-BY-4.0 允许任何人：

✅ **可以**:
- 自由下载、使用、复制项目中的数据
- 创建衍生作品或修改数据
- 在任何媒体或格式中重新发布或改编数据
- 用于商业目的

❌ **条件**:
- **必须署名**: 必须向原创作者、贡献者和许可方清晰标注
- **保持一致**: 衍生作品应采用相同或兼容的许可证

❌ **禁止**:
- 在没有署名的情况下使用
- 添加额外的法律限制（如 DRM）

## 署名要求

### 最小署名格式

使用本项目数据时，请采用以下至少之一的形式进行署名：

#### 1. 完整署名（推荐）

```
数据来源: 中国历史全视界 (Chinese Historical Panorama)
许可证: CC-BY-4.0
链接: https://github.com/your-org/chinese-historical-panorama
访问日期: [日期]
贡献者: [贡献者名单或 GitHub 用户名]
```

#### 2. 学术引用

```bibtex
@dataset{chinese_historical_panorama_data,
  title={Chinese Historical Panorama: Historical Data Collection},
  author={Contributors},
  year={2025},
  url={https://github.com/your-org/chinese-historical-panorama},
  license={CC-BY-4.0},
  note={Accessed: [日期]}
}
```

#### 3. 内联署名（网页/应用中）

```html
数据来源：<a href="https://github.com/your-org/chinese-historical-panorama">
中国历史全视界</a> (CC-BY-4.0)
```

### 署名中应包含的信息

- 原始项目名称与链接
- 许可证类型与链接（如 https://creativecommons.org/licenses/by/4.0/）
- 原始贡献者或数据来源（如 "Contributors to Chinese Historical Panorama"）
- 访问日期或版本号（可选但推荐）
- 你所进行的修改描述（如有）

## 第三方数据与混合许可

### 已整合的第三方数据源

本项目整合了来自多个开放数据源的信息，每个源都有其自身的许可。在使用这些数据时，请**同时遵守原始源的许可要求**：

| 数据源 | 内容 | 原始许可 | 署名要求 |
|--------|------|--------|--------|
| **CHGIS v6.0** | 中国历代行政区划边界 | [CHGIS License](http://www.fas.harvard.edu/~chgis/data/v6/) | 必须署名 Harvard CHGIS Project |
| **OpenHistoricalMap** | 历史地理特征 | ODbL 1.0 | 必须署名 OpenHistoricalMap contributors |
| **David Rumsey Map Collection** | 历史地图扫描件 | 各不同（见原网站） | 需验证单个地图许可 |
| **社区贡献** | 用户提交的事件、人物、地图 | CC-BY-4.0（默认） | 署名贡献者 |

### 使用混合许可数据时的规则

如果你的衍生作品包含来自多个源的数据，应当：

1. **分别列举来源**: 清晰标注每部分数据的来源与许可
2. **遵守最严格的许可**: 若混合的许可要求不同，采用最严格的
3. **示例**:
   ```
   本研究使用了以下数据：
   - 疆域边界：Harvard CHGIS (CC-BY-SA-3.0)
   - 事件信息：中国历史全视界 (CC-BY-4.0)
   - 人物资料：自有数据库 (CC-BY-4.0)
   
   因此本衍生作品采用 CC-BY-SA-4.0 许可（最严格）。
   ```

## 数据质量与准确性声明

### 免责声明

本项目的数据由多位贡献者提供，虽经核实，但**不保证**100% 准确。

我们提供以下信息：
- **置信度分数**: 每条数据包含 0-1 的置信度评分
- **数据来源**: 每条数据都指向原始来源以供独立验证
- **更新日期**: 数据的最后更新时间

### 数据使用者的责任

使用本项目数据的研究者或开发者应当：
1. **独立验证**: 关键数据应通过原始学术文献验证
2. **理解限制**: 了解数据的不确定性与可能的偏差
3. **保持透明**: 在自己的工作中说明数据来源与任何修改

## 常见问题

### Q: 我能在我的商业应用中使用这个数据吗？
**A**: 是的，完全可以。CC-BY-4.0 允许商业使用，只需在你的应用中清晰署名。

### Q: 我需要经过批准才能使用数据吗？
**A**: 不需要。CC-BY-4.0 是自动许可，你不需要单独请求权限，只需遵守署名要求。

### Q: 如果我修改了数据，我的版本也需要使用相同许可吗？
**A**: CC-BY-4.0 本身不强制衍生作品采用相同许可。但如果你想将修改回馈给社区，我们建议采用相同或兼容的许可。

### Q: 我能把数据纳入我的商业数据库并出售吗？
**A**: 可以，但必须：
1. 署名原始来源
2. 明确说明数据来自 CC-BY-4.0 许可的开放项目
3. 不添加额外的法律限制

### Q: 数据集的哪些部分受许可约束？
**A**: 以下部分受 CC-BY-4.0 约束：
- 事件记录与描述
- 人物传记与年表
- 地理边界与坐标数据
- 原始数据编译与清理工作

但不包括：
- 代码（仍采用 MIT）
- 项目文档（推荐使用相同许可）
