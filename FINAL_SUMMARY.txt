================================================================================
  �� 项目全景概览
================================================================================

项目名称:        中国历史全视界
英文名称:        Chinese Historical Panorama

核心功能:
  1. 多维筛选: 时间轴、事件、人物、地点四维联动
  2. 历史地图: 基于时间轴的中国疆域与行政区划演变可视化
  3. 交互平台: Web 应用 + 开源数据库 + 社区贡献机制
  4. 学术引用: 完整的数据来源追踪与学术规范

目标用户:
  - 历史研究者与高校师生
  - 地理信息专业人员
  - 历史爱好者与内容创作者
  - 教育工作者与学生

项目规模:
  - 开发时间: 6-8 周 (单人) / 4-6 周 (3-4 人) / 2-3 周 (5-7 人)
  - 预计代码行数: ~15K (Phase 0 完成时)
  - 预期贡献者: 5-10 人 (第 2 季度)

================================================================================
  🛠️ 技术栈决策 (已确认)
================================================================================

前端:
  ⭐ React 18 + TypeScript        (声明式 UI，类型安全)
  ⭐ MapLibre GL + Deck.gl        (矢量瓦片 + GPU 加速)
  ⭐ Vite + Tailwind CSS          (快速开发与响应式设计)
  ⭐ Zustand                      (轻量级状态管理)

后端:
  ⭐ Fastify + Node.js 18         (高性能 API，成熟生态)
  ⭐ Prisma ORM                   (类型安全的数据库操作)
  ⭐ PostgreSQL 15 + PostGIS 3.3  (关系数据库 + 地理空间扩展)

地理数据:
  ⭐ GeoJSON / TopoJSON           (标准数据格式)
  ⭐ GDAL/OGR + Tippecanoe        (数据处理与矢量瓦片生成)
  ⭐ DVC + Git LFS                (版本控制与大文件管理)

部署:
  ⭐ GitHub Actions               (CI/CD 自动化)
  ⭐ Netlify / Vercel             (前端静态部署)
  ⭐ Render / DigitalOcean        (后端容器化部署)
  ⭐ Cloudflare + S3/OSS          (CDN + 对象存储)
